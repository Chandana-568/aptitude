# Number System Formula Sheet

## 1. Even and Odd Numbers

### Even Number

* Divisible by 2
* Form:


  2n
  

### Odd Number

* Not divisible by 2
* Form:

  ```
  2n + 1
  ```

### Properties

| Operation   | Result |
| ----------- | ------ |
| Even + Even | Even   |
| Odd + Odd   | Even   |
| Even + Odd  | Odd    |
| Even × Even | Even   |
| Even × Odd  | Even   |
| Odd × Odd   | Odd    |

# 3. Sum Formula

First n Natural Numbers

[
\frac{n(n+1)}{2}
]

---

First n Even Numbers

```
n(n+1)
```

---

First n Odd Numbers

```
n²
```

---

Squares

```
1²+2²+3²+...+n²

=

n(n+1)(2n+1)/6
```

---

Cubes

```
1³+2³+3³+...+n³

=

[n(n+1)/2]²
```

# 4. Divisibility Rules

| Number | Rule                                                       |
| ------ | ---------------------------------------------------------- |
| 2      | Last digit even                                            |
| 3      | Sum of digits divisible by 3                               |
| 4      | Last two digits divisible by 4                             |
| 5      | Ends with 0 or 5                                           |
| 6      | Divisible by both 2 and 3                                  |
| 8      | Last three digits divisible by 8                           |
| 9      | Sum of digits divisible by 9                               |
| 10     | Ends with 0                                                |
| 11     | Difference between odd and even place sums divisible by 11 |
| 12     | Divisible by 3 and 4                                       |

# 21. Base Conversion

### Decimal → Binary

Repeatedly divide by **2** and record the remainders.

### Binary → Decimal

Multiply each bit by the corresponding power of **2** and add the results.
